# RepoJifrex

Repositorio de experimentos de código para **Jifrex Studio** — Software as a Structure.

## Stack
- TouchDesigner (generative visuals)
- MaxMSP / RNBO (audio interactivo)
- Ableton (producción y performance)
- AWS (infraestructura cloud)
- MCP Servers (laboratorios de integración IA)

## Estructura
```
RepoJifrex/
├── plugins/          # Plugins Max for Live, VST, RNBO exports
├── touchdesigner/    # Patches y componentes TD
├── maxmsp/           # Patches MaxMSP
├── mcp-labs/         # Experimentos con MCP servers
└── aws/              # Scripts y configs de infraestructura
```

## Filosofía
El software como material artístico. Cada experimento es una estructura.
